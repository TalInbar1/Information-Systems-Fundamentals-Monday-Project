# Information Systems Fundamentals Project
## System Characterization & Implementation via Monday.com

### Project Overview
This project involves the comprehensive characterization, analysis, and design of an information system, including a functional implementation using the **Monday.com** platform. The goal was to solve organizational challenges through structured data management and workflow automation.

### Key Features & Contributions:
* **System Characterization:** Defining functional and non-functional requirements, user personas, and use-case scenarios.
* **Process Automation:** Implementing automated workflows in Monday.com to streamline operations and reduce manual errors.
* **Architecture Design:** Creating DFD (Data Flow Diagrams) and ERD (Entity Relationship Diagrams) to support system logic.
* **Platform Implementation:** Building customized boards, dashboards, and integration logic within a low-code environment.

### Project Structure:
* `Part 1:` Focuses on system requirements and organizational analysis.
* `Part 2:` Covers system design, data modeling, and the final implementation details.
